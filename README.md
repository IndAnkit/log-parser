# log-parser

Log Parser is a MERN app for dealing with log files.

## Installation and run

Use this command to clone and run.

```bash
#clone the Repo
git clone 


#
cd log-parser-client

#instaling node package
npm i 

# Run client
npm start

# Run this command inside root folder
cd server

#instaling node package
npm i 

#run server
node index.js

```
